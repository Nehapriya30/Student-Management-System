# Student Management System

A menu-driven Student Management System developed using Java and Object-Oriented Programming principles.

## Features

* Add Student
* Display All Students
* Search Student by ID
* Update Student Details
* Delete Student
* Exit Application
* Invalid Input Handling

## Technologies Used

* Java
* ArrayList
* OOP Concepts
* Scanner Class

## Project Structure

* Student.java
* Main.java

## Concepts Practiced

* Classes and Objects
* Constructors
* Methods
* ArrayList
* Loops
* Switch Statements
* CRUD Operations

## How to Run

1. Compile the files:
   javac Main.java Student.java

2. Run the application:
   java Main
